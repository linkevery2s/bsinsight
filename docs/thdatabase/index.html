  • このサービスについて

      「台風○号が勢力を拡大し、接近中です。」「猛烈な」「大型の」昨今の日本に上陸する台風は、様々な表現がされております。

      「今度来る台風は過去の台風と比べて規模はどのくらいなのだろう？」「似たような経路の台風はないのかな？」

      本データベースは、そのような台風の統計資料となっております。平成13年（2001年）以降に発生した台風のうち、日本に上陸した台風をまとめました。

      中心気圧や風速など、フィルターをかけて並べ替えをしたり、キーワード検索をしたり、台風の経路シミュレーションマップを見たり、使い方は様々です。

      ぜひ、一度ご覧いただければと思います。仁志.

  • 使い方

      キーワード検索で号数などから該当する台風を抽出することができます。

      先頭列（中心気圧や風速など）をクリックするとソート順（昇順、降順）に並び替えが可能です。

      マップをクリックすると台風経路図がマップに表示されます。

  • 用語定義

      中心気圧

      各台風における中心気圧（hPa）の最も高いものを掲載しています。

      風速

      各台風における最大瞬間風速（m/s）を掲載しています。

      暴風域

      各台風における暴風域の最大半径（km）を掲載しています。

      強風域

      各台風における強風域の最大半径（km）を掲載しています。
