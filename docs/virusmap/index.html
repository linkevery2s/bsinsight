はじめに

新型コロナウイルスの感染が世界中に広がっています。不安が広がってやまないですが、退院される方も増えてきていることは事実です。刻一刻と変わるウイルスの感染状況に対し、感染者数などの絶対数ではなく、週ごとに感染の流れを「動態」という形で、ウイルスの特徴などを探れないかと思い、このWEBを立ち上げました。
